# CandorLoop demo script — target 2:35

The video must remain under three minutes. Record at 1080p if possible, keep the pointer movement slow, and show the **Live GPT-5.6** badge during the main run.

## 0:00–0:12 — Title and problem

**On screen:** CandorLoop hero and tagline.

**Voiceover:**  
“AI can be helpful and still become an echo—agreeing with the conclusion we want, or losing the goal during a long conversation. CandorLoop makes that drift visible.”

## 0:12–0:28 — Product promise

**On screen:** Slowly scroll from the hero to the three scenarios.

**Voiceover:**  
“The idea is simple: keep the goal, test the judgment, and explain the change. The human stays in control.”

## 0:28–0:52 — Set the course

**On screen:** Select **The rushed launch**. Briefly highlight the Main goal and Conversation fields.

**Voiceover:**  
“Here the real goal is a reliable launch that protects customer trust. But the latest request asks for a confident announcement even though a payment issue is unresolved.”

## 0:52–1:18 — Run GPT-5.6

**On screen:** Press **Run an independent read**. Wait for the result. Pause on the **Live GPT-5.6** badge, goal-fidelity ring, and agreement-pressure value.

**Voiceover:**  
“GPT-5.6 returns a structured analysis—not a block of persuasive text. CandorLoop separates user intent from independent assessment, scores alignment with the goal, and identifies pressure for automatic agreement.”

## 1:18–1:40 — Inspect the judgment

**On screen:** Point to Evidence signals, What would change this judgment, and Helpful final response.

**Voiceover:**  
“It shows the observable signals, the evidence that would change the judgment, and a response that is candid without being dismissive. It proposes a limited pilot instead of either blind approval or a dead end.”

## 1:40–1:58 — Return to the goal

**On screen:** Press **Return to main goal** and show the refreshed assessment and Position history.

**Voiceover:**  
“If the conversation drifts, Return to main goal finds the smallest useful course correction. A position can change, but the reason has to be legible.”

## 1:58–2:18 — Protect creativity

**On screen:** Select **Let the idea breathe**, run the analysis, then press **Let It Finish**.

**Voiceover:**  
“Candor is not premature criticism. Let It Finish first gives a raw idea its strongest coherent form, then separates assumptions from facts and identifies the smallest meaningful test.”

## 2:18–2:35 — Close

**On screen:** Scroll to The Candor Loop, then return to the logo/tagline.

**Voiceover:**  
“CandorLoop is a complete Next.js prototype built with Codex and the GPT-5.6 Responses API using Structured Outputs. AI that doesn’t just agree—and doesn’t take the decision away from you.”

## Recording checklist

- Use a fresh browser window with no personal bookmarks, notifications, or account information visible.
- Confirm the API key is configured and the result badge says **Live GPT-5.6**.
- Keep browser zoom at 90–100% so the dashboard reads clearly.
- Do one silent rehearsal and one timed rehearsal before recording.
- Narrate how both Codex **and** GPT-5.6 were used.
- Export below three minutes and upload as a public or unlisted YouTube video accepted by the rules.
